# Fernet-Encoder-Decoder
A simple encoder + decoder for fernet keys 


For Linux
1. Open Terminal
2. Git clone https://github.com/Dyablows1309/Fernet-Encoder-Decoder.git
3. Cd Fernet-Encoder-Decoder
4. Pip install -r requirements.txt
5. Python enc.py


For windows:
1. Download the file and place in a folder
2. Open cmd navigate to the folder
3. pip install -r requirements.txt
4. py enc.py
5. have fun
